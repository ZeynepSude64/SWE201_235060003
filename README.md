# SWE201_235060003
This project is a simple C# Console Application that demonstrates:
- Class usage
- Properties
- Constructors (empty and parameterized)
- Member function
- Object creation and method calling


## Files
- **Program.cs**: Contains the Main method
- **Weather.cs**: Contains the Weather class with properties, constructors, and functions


## How it works
1. Two Weather objects are created.
2. Their information is displayed using the Describe() function.


This project follows all required rules of the assignment.
